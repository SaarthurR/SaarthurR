## Hi, I'm Saarth 👋

I build full-stack web apps, AI tooling, and hardware dashboards — usually the whole thing end to end, from the serverless backend to the interface. Lately that's meant robotics control panels, LLM-powered outreach and vision tools, and community apps that real people use.

- 🤖 **Robotics & hardware** — web dashboards for robot arms and tactile-sensing gloves (live serial/CAN telemetry, 3D visualization in the browser)
- 🧠 **AI tooling** — GPT-4o vision, LLM draft generation, prompt engineering, and agent workflows
- 🌐 **Full-stack** — Next.js · React · Supabase · Firebase · Drizzle · deployed on Vercel
- 📈 Also into systematic trading and Pine Script on the side
