I build the whole thing end to end — from the serverless backend to the interface. Lately that's meant robotics control panels, LLM-powered vision and outreach tools, and community apps that real people use.

- 🤖 **Robotics & hardware** — web dashboards for robot arms and tactile-sensing gloves (live serial/CAN telemetry, 3D visualization in the browser)
- 🧠 **AI tooling** — GPT-4o vision, LLM agents, prompt engineering, MCP servers
- 🌐 **Full-stack** — Next.js · React · Supabase · Firebase · Drizzle · Vercel
- 📈 Systematic trading and Pine Script on the side

### Featured work

| Project | What it is |
|---------|------------|
| [**Juber** — JCNC Carpool](https://github.com/SaarthurR/Juber) | Google-auth carpool board for a temple community — post/find rides, realtime seat coordination, event ride boards. Next.js + Supabase. |
| [**Ascenta 3**](https://github.com/SaarthurR/ascenta3) · [live](https://ascenta3.vercel.app) | Session-gated arcade portal — access-key login, HMAC-signed sessions verified at the edge. Vercel + Firebase. |
| [**Outreach CRM**](https://github.com/SaarthurR/outreach-crm-dashboard) | Lead discovery + AI-personalized email drafts + Gmail sending + reply tracking via Pub/Sub. Next.js + Drizzle. |
| [**Diggity**](https://github.com/SaarthurR/Diggity) | Snap an artifact, GPT-4o vision returns a structured archaeological dossier. React 19 + Express + OpenAI. |
| [**Tactile Glove Viewer**](https://github.com/SaarthurR/tactile-glove-demo-linux) | Offline browser viewer for a tactile glove — live finger bend, pressure heatmap, 3D hand over USB. |
| [**saarth-site**](https://github.com/SaarthurR/saarth-site) | Personal site — Three.js particles, GSAP animation, and a playable tabla. |
